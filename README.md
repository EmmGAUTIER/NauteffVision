# NauteffVision
Maquette de logiciel d'affichage de données de navigation.

NauteffVIsion permettra de collecter des données de navigationi, de la traiter, de les afficher et de les enregistrer pour traitement ultérieurs. La version proposée est une simple maquette et est destinnée à l'évaluation et la spécification.

Le fichier NauteffVision.json contient la configuration de l'afficheur.
