class data.Data(dtype='?', timestamp=None, initialFrame=None, origin=None, values=None)¶

Bases: object

Data used by NauteffVision are stored in classes that inherit from this Data. It is meant to store a raw frame, its origin, its type, a timestamp and cooked data.
